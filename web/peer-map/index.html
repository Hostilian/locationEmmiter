Location Emitter — decode & map

Paste LEP or mesh hex from serial, or build frames from this phone’s GNSS (no on-air BLE from the browser).

Android app: allow location for this app in system settings if Use my location fails.

You appear offline — map tiles may not load until you reconnect.
Help & shortcuts
  • Plot: button or Ctrl+Enter in the hex field (Mac: Cmd+Enter).
  • Focus hex: press / when focus is not in a text field (useful for keyboard-only use).
  • This panel: press ? (shift+/) to open Help from the map or page background.
  • Paste clipboard appends hex from the system clipboard (useful on Android after copying from a terminal or chat).
  • Shareable link: Copy link (or open with ?hex=… / #hex=…) to prefill and plot; very long payloads use the hash (length capped).
  • Load file accepts text logs; max about 1.5 MB per file.
  • Export: GPX / CSV / GeoJSON use the last plotted decode markers plus the encode pin if shown. On Android, the system share sheet opens when the browser supports file sharing; otherwise a download runs.
  • Theme: Auto follows the device; Light/Dark are saved for next visit.
  • Install: browsers can add this app to the home screen (manifest); Android also has the Capacitor build.
