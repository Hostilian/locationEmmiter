# Market Validation & Competitive Analysis

## 1. Competitive Landscape
| Feature | Location Emitter | Garmin InReach | Meshtastic (DIY) |
| :--- | :--- | :--- | :--- |
| **Recurring Fee** | Optional (Pro) | Mandatory ($15+/mo) | None |
| **Ease of Use** | High (Consumer App) | Moderate (Device-centric) | Low (Technical/DIY) |
| **Mesh Networking** | Native (High Density) | None (Point-to-Sat) | Native (Community) |
| **Hardware Cost** | $149 | $350+ | $50 (DIY) |
| **E2E Encryption** | AES-256 | Proprietary | AES-256 |
